# progress-bar

Scripts that emit [graphical progress bar](https://ghostty.org/docs/install/release-notes/1-2-0#graphical-progress-bars) escape codes alongside a terminal progress bar. Each script loops 100 times with a weighted random delay, reporting percent completion via ConEmu-style OSC escape codes.

## Usage

```bash
# Ruby
ruby progress.rb

# Python
python3 progress.py

# Node.js
node progress.js
```
